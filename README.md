## 🚀 Projets phares

### 🔍 Analyse & Maintenance

#### [RCA System — Root Cause Analysis Industriel](https://github.com/wassimbelaid05-EIE/systeme-RCA)
Moteur Python d'analyse de causes racines pour équipements industriels. Combine **FTA booléenne** (IEC 61025), **FMEA avec scales AIAG** (IEC 60812), **détection d'anomalies multi-capteur** (Z-score + Isolation Forest) et **inférence bayésienne** pour ranking des causes probables. Exports Excel 7-sheets professionnels.

`Python` `scikit-learn` `pandas` `IEC 61025` `IEC 60812` `ISO 9001`

#### [RCA Dashboard React](https://github.com/wassimbelaid05-EIE/rca-dashboard-react)
Interface web premium **dark-obsidian** pour visualisation RCA. Design professionnel avec fonts Syne + DM Mono, 5 onglets interactifs (FTA, FMEA, Sensors, Causes, Actions), exports Excel et HTML/PDF. Déployé sur Vercel.

`React 18` `Vite` `Recharts` `Lucide Icons` `Vercel`

#### [Maintenance Prédictive ML](https://github.com/wassimbelaid05-EIE/maintenance-predictive-ml)
Système de prédiction de défaillances basé sur **machine learning**. Pipeline complet : collecte données capteurs, feature engineering, entraînement modèles (Random Forest, XGBoost, LSTM), déploiement avec scoring temps réel et alertes.

`Python` `XGBoost` `LSTM` `Time Series` `Predictive Analytics`

#### [CMMS — Maintenance System](https://github.com/wassimbelaid05-EIE/systeme-de-maintenance-CMMS)
GMAO complet (Gestion de Maintenance Assistée par Ordinateur) — planification PM/CM, work orders, inventaire pièces détachées, KPIs MTBF/MTTR, génération de rapports. Interface web responsive pour techniciens terrain.

`Python` `Flask` `SQLite` `Bootstrap` `Maintenance KPIs`

### ⚡ Électrotechnique & Simulation

#### [Electromagnetic Actuator FEM Simulator](https://github.com/wassimbelaid05-EIE/actionneur-electromagnetique-simulateur-FEM)
Simulateur 2D d'actionneurs électromagnétiques avec **méthode des éléments finis**. Calcule densités de flux, forces, inductances et courbes de saturation pour solénoïdes et actionneurs linéaires. Interface Streamlit pour exploration paramétrique.

`Python` `FEM` `NumPy/SciPy` `Streamlit` `Magnetostatique`

#### [Calculateur Dimensionnement Électrique](https://github.com/wassimbelaid05-EIE/calculateur-de-dimensionnement-electrique)
Outil professionnel de **dimensionnement électrique** — sections de câbles, protections différentielles, chutes de tension, courants de court-circuit. Conforme NF C 15-100 et IEC 60364.

`Python` `NF C 15-100` `IEC 60364` `Electrical Design`

#### [Simulateur de Réseau Électrique](https://github.com/wassimbelaid05-EIE/simulateur-de-reseau-electrique)
Simulateur de **flux de puissance** dans réseaux électriques — méthode Newton-Raphson, analyse de charge, calcul de pertes, études de stabilité. Support réseaux MT/BT résidentiels et industriels.

`Python` `Power Flow` `Newton-Raphson` `Grid Analysis`

#### [Commande de Moteur PID](https://github.com/wassimbelaid05-EIE/commande-de-moteur-PID)
Implémentation complète de **régulation PID** pour moteurs électriques — auto-tuning (Ziegler-Nichols), anti-windup, filtrage dérivé, simulation de charge variable. Interface graphique pour comparaison de réglages.

`Python` `Control Theory` `PID Tuning` `Simulink-like`
